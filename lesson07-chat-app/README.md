# Lesson 07 – Chat Applications (Ollama)

This exercise is part of **Lesson 07 – Building Chat Applications**
from the Microsoft *Generative AI for Beginners* course.

## Exercise
Build a simple chat application that:
- Uses chat-style messages
- Supports roles (system, user, assistant)
- Maintains conversation history

## Model
- Local LLM via Ollama
- Model: `phi3:mini`

## Run
```bash
python chat_app.py
